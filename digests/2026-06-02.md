## Notable gate/critic actions

- **Netlogon:** critic caught a true-negative BER defect — `content:"|63|"; depth:1;` can never match a CLDAP `LDAPMessage` (always starts with `0x30` SEQUENCE); fixed to anchor `0x30` at byte 0 then `0x63` via `distance:0`.
- **Red Hat Miasma:** critic dropped a bare `tls.sni:"api.anthropic.com"` rule (fires on all benign Claude-API usage; SNI can't see the `/v1/api` path).
- **WP Maps Pro:** Sigma access-log rule downgraded high→medium (body-borne exploitation is invisible to query-string access logs; the Snort/Suricata rules carry that vector).
- **TP-Link:** gated to no-detection rather than ship a broad "any request to the router" rule.
